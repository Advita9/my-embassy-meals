
# my-embassy-meals

Simple scrapbook-style webpage for documenting quick meals.

## Tech Stack

- Next.js 15
- React 19
- TypeScript
- Tailwind CSS
- Google Fonts (Newsreader, Caveat, Sacramento, Patrick Hand)
- CSS 3D Transforms for card flip animations

## Features

- Scrapbook-inspired UI
- Polaroid recipe cards
- Flip-to-view ingredient notebook pages
- Handwritten typography and paper textures
- Responsive grid layout
- Scrollable recipe notebooks for longer ingredient lists

## Getting Started

```bash
npm install
npm run dev
